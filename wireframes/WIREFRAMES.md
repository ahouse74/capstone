### 1. Display Name Prompt
First-time visitor screen that collects a display name before accessing the app.

### 2. Main View (Home)
Two-panel layout showing an interactive map with color-coded markers and a filterable grid of animal listing cards.

### 3. Listing Detail Page
Full animal profile with photo, description, location info, status update controls, and community comments section.

### 4. New Listing Form
Submission form with photo upload, animal details, species selector, description field, and click-to-place map pin.
